Clicking Load/Display:  
-User selects bmp from directory
-Image is displayed inside GUI

Clicking Histogram Equalization:
-Image is equalized and displayed in GUI

Clicking Overlay Image:
-User selects bmp from directory (Overlay text; must be same size image)
-Two images are "combined" to create overlay image
-New image is displayed inside GUI

Brightness and Contrast Sliders:
-User moves slider
-Displayed number changes after mouse is released
-Number is read as input into code
--Brightness Slider adds or subtracts values to Pixel Data
--Contrast Slider multiplies value to Pixel Data
-New Image is displayed inside GUI

All new images are saved as bmp files.
